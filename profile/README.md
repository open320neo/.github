# Willkommen bei Open320Neo ✈️

Willkommen auf dem offiziellen GitHub-Profil von Open320Neo!

Wir entwickeln gemeinsam einen realistischen Airbus A320neo Flugsimulator — von Hardware über Software, Elektronik und Infrastruktur.

## 🔧 Was wir machen
- Entwicklung eines A320neo Cockpits
- Eigene Hardware & Elektronik
- Integration mit Microsoft Flight Simulator
- 3D-Druck & Lasercutting
- Netzwerk- & Serverinfrastruktur
- Dokumentation & Open-Source Projekte

## 🚀 Technologien

- Microsoft Flightsimulator 2024
- MobiFlight
- Single Board Computer und Microcontroller
- 3D-Druck
- CAD & PCB Design
